Aprendendo sobre HTML

Uma breve explicação do que é HTML

HTML, sigla para HyperText Markup Language, é a linguagem de marcação utilizada para criar e estruturar conteúdo em páginas da web. Consiste em uma série de elementos e tags que definem a hierarquia e o formato do conteúdo, permitindo a criação de documentos hipertexto que podem incluir texto, imagens, links e outros elementos interativos.

As tags

Tags em HTML são instruções entre colchetes angulares usadas para definir a estrutura e formato de elementos em uma página da web.

!DOCTYPE

é uma declaração que especifica a versão do HTML sendo utilizada no documento, sendo comumente usada para indicar a conformidade com HTML5.
